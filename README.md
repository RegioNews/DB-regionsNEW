# GitHub Pages структура: RSS по 89 субъектам РФ

## Структура
- `index.html` — браузерный просмотрщик реестра
- `data/rss-subjects-rf-89.json` — JSON с 89 субъектами РФ
- `data/rss-subjects-rf-89.opml` — OPML для импорта RSS
- `.github/workflows/deploy.yml` — автодеплой на GitHub Pages

## Как использовать
1. Загрузите содержимое папки в GitHub-репозиторий.
2. Включите GitHub Pages.
3. После публикации откройте сайт и JSON/OPML по прямым URL внутри репозитория.
